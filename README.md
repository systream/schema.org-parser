# schema.org-parser
Schema.org microformat parser

## Build

* ```npm install```
* ```grunt build```

## usage

You should include the schema.org parser.

```<script src="dist/schema.org-parser.min.js"></script>``` 

And than simply run: 
```javascript
var microformats = getMicroFormats();
```
